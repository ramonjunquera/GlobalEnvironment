qDebug

Novedades:
- Librería QDebug.h

Descripción:
  Cuando queremos queremos mostrar un mensaje por la consola utilizamos el comando cout.
  Para ello se debe incluir la librería iostream.h y usar el espacio de nombres std.
  Esto es correcto.
  Para incluir mensajes de debug también podemos hacerlo, pero hay una librería
  específica para ello llamada QDebug.h.
  La sintaxis es muy similar. Sólo hay que sustituir cout por qDebug().
  No es necesario endl. Le pone un retorno de carro automáticamente a cada mensaje.
  Se pueden encadenar distintas variables.
  Tiene algunas limitaciones respecto a cout porque no es capaz de representar todos los
  tipos de variables. En esos casos se requerirá conversión.
Desarrollo:
  Se muestra un programa muy simple que crea mensajes de debug.
