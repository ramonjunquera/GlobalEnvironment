HelloWorld

Novedades:
- Primera aplicación de línea de comandos
- Escritura en pantalla

Funcionalidad:
- Se muestra un texto en pantalla

Desarrollo:
  Es la primera vez que se crea una aplicación de línea de comando (sin entorno gráfico).
  A la hora de crear un nuevo proyecto en QtCreator, seleccionamos:
    Projects/Application/Qt Console Application
  Se crea la estructura de una aplicación simple.
  Sólo contiene la función main.
  Queremos escribir un texto en pantalla con el comando cout.
  Pertenece a la librería iostream por eso debemos añadirla en la sección de includes.
  También indicamos que utilizaremos el espacio de nombres estándar con:
    using namespace std;
