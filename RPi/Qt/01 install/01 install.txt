Tema: Instalación de Qt5 en Linux
Fecha: 20220318
Autor: Ramón Junquera

Para Raspberry Pi.
No existe un paquete descargable desde la página oficial de Qt que esté compilado para ARM.
Pero sí tenemos uno preparado para descargar desde los repositorios de Raspberry.

Desde una sesión de Terminal ejecutamos:
  sudo apt-get install qt5-default
    Nota. En las últimas versiones ya no está disponible. Se puede instalar su contenido
    manualmente así:
      sudo apt-get install qtbase5-dev qtchooser qt5-qmake qtbase5-dev-tools
  sudo apt-get install qtcreator
Podemos comprobar la versión instalada de qtcreator con:
  apt-cache show qtcreator
A fecha actual es la:
  Version: 4.14.1-1

Si tenemos intención de utilizar el módulo de multimedia, para crear aplicaciones
que reproduzcan audios o videos, además hay que añadir las siguientes librerías:
  sudo apt-get install qtmultimedia5-dev libqt5multimediawidgets5 libqt5multimedia5-plugins libqt5multimedia5

Ejecución:
- Desde terminal con: qtcreator
- Desde menú Programming/Qt Creator

Nota importante!!!
Para poder utilizar las distintas interfases del chip BCM2835 en Qt es obligatorio activarlas!
Lo podemos hacer desde:
- El terminal con: sudo raspi-config
  Seleccionamos "3 Interface Options" y aquí activamos todas las que necesitemos.
- Desde Menú/Preferences/Raspberry Pi Configuration/Interfaces
Si no lo hacemos, los programas que desarrollemos con Qt pueden fallar.
Ejemplo: aplicación que gestiona I2C falla al hacer una lectura (no escritura!).
Con aplicaciones desarrolladas bajo el entorno Arduino de Platformio, no es obligatorio
activar las interfases.


Para Ubuntu en un PC (x64).
Descargar la última versión oficial completa de la página oficial:
  https://download.qt.io/official_releases/qt/
Actualmente, la última es:
  https://download.qt.io/official_releases/qt/5.12/5.12.10/
La versión para Linux es:
  https://download.qt.io/official_releases/qt/5.12/5.12.10/qt-opensource-linux-x64-5.12.10.run

Le proporcionamos permisos de ejecución.
  Mediante línea de comandos: chmod +x ~/Downloads/qt-opensource-*
  O a través del explorador,. propiedades y marcando el check de hacer el archivo ejecutable.
Lo ejecutamos desde el explorador con doble click.

Para instalarlo necesitaremos una cuenta válida de Qt. Son gratuitas.

Marcamos todos los componentes que queramos utilizar.
