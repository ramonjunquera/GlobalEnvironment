Cómo instalar el IDE de Arduino en Raspbian ARM32 o x86
Fecha:20210428

Descargamos la última versión desde la página oficial:
https://www.arduino.cc/en/Main/Software
Suponemos que quedará en la carpeta Downloads del usuario pi (/home/pi/Downloads).

Actualmente la última versión es la 1.8.13.
Dependiendo del tipo de procesador obtenemos el archivo:
  ARM32  : arduino-1.8.13-linuxarm.tar.xz
  Linux64: arduino-1.8.13-linux64.tar.xz

Descomprimimos el archivo en /usr/share para que quede a disposición de todos los usuarios
  sudo tar -Jxvf /home/pi/Downloads/arduino-1.8.13-linuxarm.tar.xz -C /usr/share
  sudo tar -Jxvf /home/pi/Downloads/arduino-1.8.13-linux64.tar.xz -C /usr/share
Ejecutar el script install con permisos de root:
  sudo /usr/share/arduino-1.8.13/install.sh
Indica que está añadiendo un enlace al escritorio y trabajando en las asociaciones de archivos.
Tras un rato finaliza. Tenemos un nuevo icono en Menú/Programming/Arduino IDE y en el escritorio.

En home del usuario pi se ha creado la carpeta Arduino/libraries donde se guardarán las librerías
comunes.

Después de reiniciar, ya tenemos asociadas las extensiones con la aplicación.

Nota.
En Ubuntu, si utilizamos puertos de conexión especiales (/dev/ttyACM0 o /dev/ttyS0), ya
sea porque usamos placas que lo necesitan o porque tenemos la máquina virtualizada, es
posible que nos encontremos
con errores a la hora de transferir el compilado por falta de permisos.
Esto se corrige dando permisos globales al puerto:
  sudo chmod a+rw /dev/ttyACM0
  
Desinstalación:
  sudo /usr/share/arduino-1.8.13/uninstall.sh
  sudo rm -rf /usr/share/arduino-1.8.13
