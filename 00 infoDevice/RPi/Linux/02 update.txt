Tema: Actualización de sistema

Se puede forzar la actualización de sistema operativo en Raspbian con:
  sudo rpi-update
Nota: No es recomendable. No se considera versión estable. Podría dar problemas.

Para cualquier otra versión de Linux.
Actualizamos librerías y sistema operativo:
  sudo apt-get update
  sudo apt-get upgrade
Es conveniente eliminar los paquetes no utilizados:
  sudo apt-get autoremove

Es posible que se deba reiniciar tras las actualizaciones
