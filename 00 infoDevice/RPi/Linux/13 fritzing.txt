Tema: Cómo instalar Fritzing
Fecha: 20210419

Nota.
No utilizaremos la web oficial de Fritzing para la descarga.
Usaremos paquetes instalables por apt-get.
La razón es que la web oficial solicita registro para la descarga y además la aplicación
requiere de librerías obsoletas que no se incluyen.

Instrucciones:
  Instalación.
  - Abrimos terminal e instalamos:
      sudo apt-get install fritzing-parts fritzing-data fritzing
  - Tras la instalación se crea un icono en la sección Programming.
  - También se puede ejecutar desde la línea de comandos con: fritzing.
  Asociación de extensión.
    Raspbian
      - Hacemos click con el botón derecho sobre un archivo con extensión fzz
      - Seleccionamos "Properties"
      - Desplegamos el campo "Open with:"
      - Seleccionamos Programming/Fritzing
      - Marcamos el checkbox de "Set selected application as default..."
      - Pulsamos el botón OK
      - Fritzing se abre (pero no con el archivo indicado).
    Ubuntu
      - Localizamos un archivo con extensión .fzz (por ejemplo Mega.fzz)
      - Abrimos una sesión de terminal en esa carpeta
      - Creamos una nueva asociación con:
          mimeopen -d Mega.fzz
      - Pide seleccionar una opción. Seleccionamos la opción "Other..."
      - Escribimos el comando de asociación:
          /usr/bin/fritzing %f
      - A partir de ahora se abrirá la aplicación Fritzing cuando se haga doble click sobre 
        un archivo .fzz
  Abrir desde explorador de archivos
    Debido a un error en el script de inicio, cuando se hace doble click sobre un archivo
    con extensión fzz desde el explorador de archivos, se abre la aplicación, pero nunca
    el archivo indicado.
    Lo corregiremos con:
      Raspbian: sudo mousepad /usr/bin/fritzing
      Ubuntu  : sudo gedit /usr/bin/fritzing
    En el script vemos que primero cambia a la carpeta de la aplicación y después lanza
    la aplicación.
    Debemos añadir después del nombre del ejecutable el parámetro "$1" separado por un espacio.
      Ejemplo: exec /usr/bin/fritzing.real "$1"
    Guardamos y a partir de ahora no sólo podremos abrir archivos desde el explorador de archivos,
    sino que el path o nombre del archivo podrían contener espacios.
