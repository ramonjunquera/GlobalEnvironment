Autor: Ramón Junquera
Tema: Descripción de placa M5Stick C

Dispositivo basado en placa ESP32 (ESP32-PICO-D4) de Espressif, con 4Mb de memoria.
Conexión USB-C para puerto serie y alimentación.

Puerto serie:
  UART0 (USB-C) : Velocidad máxima 750000 baudios
    pin 1 : TX
    pin 3 : RX

Puertos
  GROOVE : I/O
    pin GND
    pin 5V
    pin 32
    pin 33
  FRONT : I/O
    pin GND
    pin 5V
    pin 26
    pin 36
    pin  0
    pin BAT
    pin 3V3
    pin 5V

Botones
  pin 39 : SIDE
  pin 37 : M5

Display
  Tamaño: 0.96"
  Tipo: TFT LCD
  Resolución: 80x160
  Modelo: ST7735S
  Color: SI
  Conexión: SPI
  Pines:
    pin  5 : CS
    pin 23 : DC
    pin 18 : RST

Batería:
  Tamaño: 80mAh
  Gestor: AXP192
  Apagar: pulsar botón POWER por 6 segundos
  Encender : pulsar botón POWER por 2 segundos

IMU
  Modelo: SH200Q
  Funciones: giróscopo y acelerómetro
  Grados de libertad (DOF): 6

SPI
  Pinout común
    pin 15 : MOSI
    pin 13 : CLK
    pin ?? : MISO

Led
  Tipo: led standard
  Color: rojo
  Pin 10
  Modo: HIGH=OFF,LOW=ON

Micrófono
  Modelo: SPM1423
  Pin 34

Buzzer: no tiene

Pines libres
  pin 32
  pin 33
  pin 26
  pin  0

I2C
  Pinout
    pin 21 : SDA
    pin 22 : SCL
  Identificadores
    0x68 : MPU6886 : acelerómetro + giróscopo (6DOF) compatible con MPU6050
    0x34 : AXP192 : gestor de batería
    0x51 : BM8563 : reloj con hora real (RTC)
