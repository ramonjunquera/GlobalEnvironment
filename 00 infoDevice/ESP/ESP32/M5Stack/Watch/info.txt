Autor: Ramón Junquera
Tema: Descripción de placa M5Stick Watch

Dispositivo basado en placa ESP32 de Espressif, con 4Mb de memoria.
Conexión USB-C para puerto serie y alimentación.

Puerto serie:
  UART0 (USB-C) : Velocidad máxima 750000 baudios
    pin 1 : TX
    pin 3 : RX

Puertos
  A : I/O
    pin 13
    pin 25
    pin 5V
    pin GND

Botones
  pin 35 : A

Display
  Modelo: SH1107
  Color: NO
  Resolución: 64x128
  Conexión: SPI
  Pinout
    pin 14 : CS
    pin 27 : DC
    pin 33 : reset

SPI
  Pinout común
    pin 18 : SCK
    pin 19 : MISO
    pin 23 : MOSI
  Pines CS
    pin 14 : display SH1107

Speaker/Buzzer
  Tipo: pasivo
  Pin 24

Led
  Tipo: led standard
  Color: azul
  Pin 19
  Modo: HIGH=ON,LOW=OFF

IR
  pin 17

Pines libres
  pin 25
  pin 13

I2C
  Pinout
    pin 21 : SDA
    pin 22 : SCL
  Identificadores
    0x68 : MPU6050 : acelerómetro + giróscopo (6DOF)
    0x0C : AK8963 : magnetómetro (3DOF)
    0x75 : IP5306 : gestor de batería
