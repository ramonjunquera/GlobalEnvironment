Tema: Cómo instalar Fritzing en Ubuntu
Fecha: 20190708

Instrucciones:
- Descargamos el paquete de instalación para linux directamente de la web oficial
  de Fritzing: http://fritzing.org/download/?donation=0
  En este ejemplo utilizaré la última versión disponible de 32bits (0.9.3b).
  El archivo descargado es fritzing-0.9.3b.linux.AMD64.tar.bz2
  Suponemos que se ha descargado en la carpeta Downloads
- Abrimos una sesión de terminal en la carpeta de la descarga
- Descomprimimios el archivo en con:
    tar -vxf fritzing-0.9.3b.linux.AMD64.tar.bz2
- Movemos la carpeta a /usr/share/fritzing con:
    sudo mv fritzing-0.9.3b.linux.AMD64 /usr/share
- Ejecutamos el script de instalación con:
    /usr/share/fritzing/install_fritzing.sh

Nota. Asociar extensión.
Habitualmente no se queda asociada la extensión .fzz con la aplicación Fritzing.
Podemos hacerlo manualmente:
- Localizamos un archivo con extensión .fzz (por ejemplo Mega.fzz)
- Abrimos una sesión de terminal en esa carpeta
- Creamos una nueva asociación con:
    mimeopen -d Mega.fzz
- Pide selaccionar una opción. Seleccionamos la opción "others"
- Escribimos la asociación:
    /etc/share/fritzing/Fritzing '%f'
