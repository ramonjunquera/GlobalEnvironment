Instalación en máquina virtual

Raspbian corresponde a un Linux Debian 8.x
Los 20Gb sugeridos son suficientes.

Instalación de vmware tools.
Ventajas:
- Cambio de resolución automática
- Portapapeles común
Pasos:
En el menú de VW Workstation. VM/Install vmware tools
Se mapea el CD a una imagen interna para la versión linux de las vmware tools.
Copiamos el archivo VMwareTools-10.0.0-2977863.tar.gz a /home/pi con el explorador de archivos
Descomprimimos allí mismo
Se crea una carpeta llamada vmware-tools-distrib
Ejecutaremos el script instalador con permisos de root con:
  sudo /home/pi/vmware-tools-distrib/vmware-install.pl
Aunque sugiera utilizar las tools de Internet, seguiremos eligiendo las del CD.
Aceptaremos por defecto a todas las opciones que nos pregunte.
Una vez finalizado, haremos caso a las sugerencias, ejecutando manualmente las tools:
  /usr/bin/vmware-user
Ya comprobamos que la resolución se adapta a la ventana.
Reiniciamos y comprobamos que los cambios se mantienen.
