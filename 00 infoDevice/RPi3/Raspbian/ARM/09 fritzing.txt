Tema: Cómo instalar Fritzing
Fecha: 20190602

La última versión de Fritzing es la 0.9.3b. Tanto para x86 como para ARM.


ARM
---
Realizamos la instalación con:
  sudo apt-get install fritzing-parts
  sudo apt-get install fritzing-data
  sudo apt-get install fritzing
Tras la instalación ya se asocia la extensión .fzz con la aplicación y se crea un icono
en la sección Programming.
También se puede ejecutar desde la línea de comandos con: Fritzing (con F mayúscula).

La primera vez que se abra la aplicación Fritzing tardará un poco porque regenera la
base de datos de componentes.


x86
---
Simplemente descargamos el paquete de instalación para linux directamente de la web oficial
de Fritzing: http://fritzing.org/download/?donation=0
En este ejemplo utilizaré la última versión disponible de 32bits (0.9.3b).
El archivo descargado es fritzing-0.9.3b.linux.i386.tar.bz2
Descomprimimios el archivo en /usr/share para que quede a disposición de todos los usuarios:
  sudo tar -xjvf /home/pi/Downloads/fritzing-0.9.3b.linux.i386.tar.bz2 -C /usr/share
Ejecutamos el script de instalación:
  /usr/share/fritzing-0.9.3b.linux.i386/install_fritzing.sh
Ahora asociamos la extensión .fzz con Fritzing.
Esto se explica en el siguiente capítulo.


Nota:
Con resoluciones bajas (1024x768 o menos) Fritzing no entra en pantalla.
Es posible que a veces se quede la ventana de la aplicación con la cabecera fuera por la parte 
superior. Esto impide que podamos moverla. Existen algunos trucos para estos casos:
  Alt + F11 : pone la aplicación a pantalla completa. Incluso quita la barra de inicio.
  Alt + click con botón derecho: mueve la ventana
