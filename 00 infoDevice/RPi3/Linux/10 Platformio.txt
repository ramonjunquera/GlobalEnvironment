Instalación de Platformio

Es requisito previo imprescindible la instalación de Visual Code, puesto que
Platformio es un plugin.

Platformio tiene un segundo requisito previo, la instalación de Python.
Esto lo podemos completar con:
  sudo apt-get install python
Si este paso se omite y Python no está instalado, aparecerá un error de ejecución
de un script .py.

La instalación consiste simplemente en añadir el plugin de Platformio.
