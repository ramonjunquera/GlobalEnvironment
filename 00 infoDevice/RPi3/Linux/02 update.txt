Una vez instalado el sistema operativo es conveniente
actualizarlo a la última versión disponible en Internet.

Actualizamos las librerías específicas de Rasberry Pi
(sólo válido en Raspberry Pi):
  sudo rpi-update
Actualizamos librerías y sistema operativo:
  sudo apt-get update
  sudo apt-get upgrade
Es conveniente eliminar los paquetes no utilizados:
  sudo apt-get autoremove
