Tema: Start vs Awake
Fecha: 20220308
Autor: Ramón Junquera

Tanto Start como Awake son funciones que se llaman una sola vez y siempre antes del primer Update.
Entre ellas, o primer se ejecuta Awake y después Start.
La única diferencia es que Awake se ejecutará SIEMPRE y Start sólamente si el
Script está activado!.

Habitualmente Awake se utiliza para la inicialización de referencias entre objetos de
las que puedan depender las llamadas de Start.

Awake se llama después de que todos los objetos están preparados.
Por eso podríamos hacer una búsqueda de por etiqueta (por ejemplo).
La ejecución de Awake entre objetos es aleatoria.
